Cubature

type, public :: Cubature

This derived type holds data, weights, and abscissae for numerical integration of 2D (triangle, quadrilaterals) and 3D bodies (tetrahedrons, hexahedrons, and wedges)

Note: Cubatures for triangles and tetrahedrons use body coordinates, and the condition x1 + x2 + x3 (+ x4) = 1 permits the elimination of the final coordinate. For wedges, the first two coordinates are body, the last is ordinary.

Domains: LINear - [-1, 1] QUAdrilateral - [-1, 1] x [-1, 1] TRIangle - [0, 1] x [0, 1-x] HEXahedron - [-1, 1] x [-1, 1] x [-1, 1] TETrahedron - [0, 1] x [0, 1-x] x [0, 1-x-y] WEJ/prism - [0, 1] x [0, 1-x] x [-1, 1]
